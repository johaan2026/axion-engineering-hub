Axion Engineering Suite

A browser-based mechanical engineering toolkit built for production-quality workflows.

Mission

Axion Engineering Suite brings commonly used engineering calculators, formulas, and reference data into a single responsive web application. It is designed to feel like modern engineering software — precise, fast, and professional — not a student prototype.

Built for

  • Mechanical engineering students
  • Mechanical design engineers
  • Manufacturing engineers
  • CAD designers and robotics enthusiasts

Technology

Vanilla HTML5, CSS3, and JavaScript (ES modules). No frameworks. Deployed as a static site on Netlify from GitHub. Calculator logic, history, and export features are added incrementally across development phases.
